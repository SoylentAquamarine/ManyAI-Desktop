ManyAI Desktop

A multi-tab AI chat desktop app that routes every conversation to the best free provider — automatically.

⚡ Electron + React
🪟 Windows · Mac · Linux
🆓 Free & open source
🔑 13 AI providers

Key Features

🗂️

Multi-tab chat

Open multiple AI conversations at once, each with its own workflow type — code in one tab, brainstorm in another.

🔀

Smart routing

Automatically picks the best provider for each task type. Fallback chains try the next provider silently if one fails.

🧩

Custom workflows

Create workflows with custom system prompts, attached context files, and auto-detect keywords. Add or remove without touching code.

🎨

Image generation

Built-in image generation via Pollinations (free) and DALL·E. Images render inline and can be downloaded.

📎

File attachment

Attach scripts or documents — content is injected silently into every message. Auto-saves AI-generated code to a .tmp file.

🔌

Custom providers

Add any OpenAI-compatible API provider from the UI. No code changes needed — just a name, base URL, and model list.

💾

Save responses

Save any AI response for later. Export code directly to a file with a smart suggested filename.

⚙️

Full routing control

Configure fallback chains per workflow type — drag providers into priority order, enable or disable per model.

Get Running

Requires Node.js 18+

git clone https://github.com/SoylentAquamarine/ManyAI-Desktop.git
cd ManyAI-Desktop
npm install
npm run dev

Add your API keys under Settings → API. Most providers have a free tier — no credit card needed.
